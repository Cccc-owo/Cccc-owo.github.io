# 博客源码

Clone 到本地:

>git clone https://github.com/Cccc-owo/Cccc-owo.github.io.git --recursive

(***配合Github Actions使用***)

Hexo 版本查看:

> hexo version

部署:

> npm install

新文章:

> hexo new <文章名>

生成文件:

> hexo g

本地测试:(127.0.0.1:4000)

> hexo s

清理生成文件:

> hexo clean
